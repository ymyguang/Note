### C语言运算规则

语句运算顺序：符号优先级，结合方向

- 算术运算符结合方向：从左往右
- 赋值运算符结合方向：从右往左
- 逗号的值是最后的的值：左结合

运算之后的数据类型，当运算变量数据类型不符是，类型低的自动转换成内存占用高的，（强数据转换），在C中的强数据转换采取：舍去原则；强数据转换是中间值，不印象原来的数据

```cpp
    int/int -> int
    int/float ->float
    %的操作数是整数，符号和被除数一致
   // 所有单位运算都是从右往左算的
```
